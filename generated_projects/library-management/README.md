# Library Management System

## 1. Project Title & Brief Description
**Library Management System** – A web application that allows users to browse, borrow, and return books, while administrators can manage inventory, users, and borrowing records. Built with modern web technologies, it provides a clean UI and robust backend powered by Prisma and Node.js.

---

## 2. Tech Stack
| Layer | Technology |
|-------|------------|
| Front‑end | HTML5, CSS3, JavaScript (ES6+) |
| Back‑end | Node.js, Express |
| Database | PostgreSQL (via Prisma ORM) |
| ORM | Prisma |
| Environment | dotenv |
| Scripts | npm (Node Package Manager) |

---

## 3. Prerequisites
- **Node.js** (v18 or later) – [Download Node.js](https://nodejs.org/)
- **npm** (comes with Node.js)
- **PostgreSQL** database instance (local or remote)
- **Git** (for cloning the repository)
