# --helo: Interactive Hello World

## Description
`--helo` is a simple web project that displays an interactive "Hello World" greeting. Users can customize the greeting text and toggle between light and dark themes, all with pure HTML, CSS, and JavaScript. No server or build tools are required—just open the page in a browser.

## Features
- **Customizable Greeting** – Edit the greeting text directly from the UI.
- **Theme Toggle** – Switch between light and dark modes with a single click.
- **Responsive Layout** – Optimized for desktop, tablet, and mobile screens.
- **Zero Dependencies** – Pure client‑side implementation; no external libraries.
- **Instant Preview** – Open `index.html` in any modern browser to see changes instantly.

## Tech Stack
- **HTML5**
- **CSS3** (Flexbox, CSS variables for theming)
- **JavaScript (ES6)**
