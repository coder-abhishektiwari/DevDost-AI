## Project Overview
This repository contains a collection of reusable, responsive, and accessible CSS components and utilities. The goal is to provide a lightweight stylesheet that can be easily integrated into any web project without the need for heavy frameworks.

## Features
- **Responsive Grid System** – Flexbox‑based layout with customizable breakpoints.
- **Utility Classes** – Margin, padding, text alignment, display, and more.
- **Component Library** – Buttons, forms, cards, modals, navigation, and alerts.
- **Theming** – Simple CSS variables for primary, secondary, and accent colors.
- **Accessibility** – Focus states, ARIA-friendly components, and high‑contrast support.

## Prerequisites
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Optional: Node.js (>= 14) and npm if you want to use the build tools
