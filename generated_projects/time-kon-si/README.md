# time-kon-si

## Project Overview

**time-kon-si** is a lightweight HTML project that displays the current time in a user-friendly format. The page updates in real-time and includes optional features such as timezone selection and customizable display styles. This project is ideal for learning basic web development concepts, including:

- HTML structure and semantics
- CSS styling and responsive design
- JavaScript for DOM manipulation and timers
- Accessibility best practices

The repository contains only static assets (HTML, CSS, JavaScript) and can be served by any static web server or opened directly in a browser.
